# SourPopcornWebApp

Getting started

To run in a container:

1. Open terminal
2. Navigate to root directory
3. Run docker-compose up --build

To run locally:

1. Create '.env.local' file and add 'VITE_API_URL=https://localhost:8081'
2. Open terminal
3. Navigate to root directory
4. Run pnpm install
5. Run pnpm dev
